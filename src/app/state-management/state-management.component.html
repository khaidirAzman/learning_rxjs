State Management

NgRx is a framework for building reactive applications in Angular. NgRx is inspired by the Redux pattern - unifying the events in your application and deriving state using RxJS

The NgRx state management process is a combination of five key elements: https://www.syncfusion.com/blogs/post/angular-state-management-ngrx.aspx

  • Store
  • Selectors
  • Reducers
  • Actions
  • Effects
